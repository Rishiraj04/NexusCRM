# NexusCRM 

NexusCRM is a cloud-native multi-tenant CRM (Customer Relationship Management) platform built using React, TypeScript, Firebase Authentication, Firestore, Express.js, and TailwindCSS. The platform is designed to simulate an enterprise SaaS application capable of managing leads, customers, tasks, analytics, and organizational workflows in isolated tenant environments.

---

# Features

- Google OAuth Authentication using Firebase Auth
- Multi-Tenant SaaS Architecture
- Enterprise Analytics Dashboard
- Customer & Lead Management
- Task Workflow Tracking
- Real-Time Firestore Synchronization
- Protected Routes & Role-Based Structure
- Modern Responsive UI using TailwindCSS & ShadCN UI
- Express.js Backend Runtime
- Modular React Component Architecture
- Context API State Management
- Runtime Error Handling & Stability Improvements
